h1. Wygwam Template Links

This ExpressionEngine extension adds a “Templates” Link Type to Wygawm’s Link dialog, making it easy to link to your site’s templates.

h3. Requirements

* "Wygwam":http://pixelandtonic.com/wygwam 2.2.2 or later
* "ExpressionEngine":http://expressionengine.com 1.7.0 or later (for EE1)
* "ExpressionEngine":http://expressionengine.com 2.1.0 or later (for EE2)

h3. Installation

h4. EE1 Installation Instructions

* Upload ee1/extensions/ext.wygwam_template_links.php to system/extensions/
* Enable Wygwam Template Links in Admin → Utilities → Extensions Manager

h4. EE2 Installation Instructions

* Upload the ee2/third_party/wygwam_template_links folder to system/expressionengine/third_party/
* Enable Wygwam Template Links in Add-Ons → Extensions
